# Learning and Teaching

This section contains posts about education, tutoring experiences, SAT preparation strategies, and learning methodologies.

<Info>
Posts will be added here during the content migration. Each post will appear in the sidebar navigation.
</Info>

## Example Post Format

When adding a new post, create a file like `sat-reading-series-lessons.mdx` with this structure:

```mdx
---
title: "Lessons from Teaching My SAT Reading Series"
description: "Reflections on designing and teaching a free 8-week SAT Reading tutoring series"
---

Your content here using MDX format...
```

Then add it to `docs.json` under the "Learning and Teaching" group:

```json
{
  "group": "Learning and Teaching",
  "icon": "graduation-cap",
  "pages": [
    "blog/learning-and-teaching/placeholder",
    "blog/learning-and-teaching/sat-reading-series-lessons"
  ]
}
```
