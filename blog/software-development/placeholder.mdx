# Software Development

This section contains posts about software engineering, system design, best practices, and lessons learned from building software.

<Info>
Posts will be added here during the content migration. Each post will appear in the sidebar navigation.
</Info>

## Example Post Format

When adding a new post, create a file like `my-post-title.mdx` with this structure:

```mdx
---
title: "Your Post Title"
description: "A brief description for SEO and previews"
---

Your content here using MDX format...
```

Then add it to `docs.json` under the "Software Development" group:

```json
{
  "group": "Software Development",
  "icon": "code",
  "pages": [
    "blog/software-development/placeholder",
    "blog/software-development/my-post-title"
  ]
}
```
